# Typescript

## 类型

**Javascript数据类型：**

1. string
2. number
3. boolean
4. null
5. undefined
6. bigint
7. symbol
8. object

> 其中object包括：Array、Function、Date、Errord等

**Typescript数据类型：**

1. 上述所有Javascript数据类型
2. 六个新类型
   1. any
   2. unknown
   3. never
   4. void
   5. tuple
   6. enum
3. 两个用于自定义类型的方式；
   1. type
   2. interface
